## Responsive Image Sizing — Assignment 3
Welcome to the Responsive Image Sizing project!

## 💡 Purpose
This project was created to demonstrate:
- Understanding responsive image techniques in web design.
- Using the srcset attribute to allow the browser to select the best image size.
- Implementing the picture element for flexible image control using media queries.
- Creating responsive background images using CSS.
- Applying a CSS grid layout to position content across different screen sizes.
- Improving user experience by ensuring images adapt properly on mobile, tablet, and desktop devices.
